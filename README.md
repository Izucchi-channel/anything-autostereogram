# Anything Autostereogram

**Recommended environment:** [Google Colab](https://colab.research.google.com/github/Izucchi-channel/anything-autostereogram/blob/main/autostereogram.ipynb)

If you run this project on Google Colab, no additional dependency installation is required.

To run the application as a local script, use: `python app.py`

**Created by:** [@izucchi-tech](https://www.youtube.com/@izucci-tech)

## How to Use

Upload either an image file or a video file that you want to convert to the designated input area.

There are two file input fields, but you should upload a file to only one of them. The application will automatically detect whether the input is an image or a video.

As an optional feature, you can provide a custom noise pattern (either an image or a video) to be used for generating the autostereogram.

For best results, use a vertically elongated noise pattern source. Since the pattern is tiled horizontally across the output, taller sources generally produce better results.

# Anything Autostereogram

**推奨実行環境：** [Google Colab](https://colab.research.google.com/github/Izucchi-channel/anything-autostereogram/blob/main/autostereogram.ipynb)

Google Colabで実行する場合、追加の依存関係をインストールする必要はありません。

ローカルで実行する場合は、`python app.py`を使用します。

**制作:** [@izucchi-tech](https://www.youtube.com/@izucci-tech)

## 使用方法

変換したい画像ファイルまたは動画ファイルを、入力欄にアップロードしてください。

深度のファイルインプット2つありますが、ファイルはどちらかにアップロードしてください。画像か動画かは自動的に判別します。

オプション機能として、オートステレオグラム生成に使用するノイズパターンを、画像または動画で指定できます。

ノイズパターンには縦長の画像または動画を使用してください。パターンは出力映像内で横方向に繰り返し配置されます。
